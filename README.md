# Audiobook API with GraphQL
- 성균관대학교 2021학년도 1학기 융합서비스 디자인 팀프로젝트
- 드라마틱 오디오북 제작 웹플랫폼을 위한 Audiobook API 제공

- MariaDB 서버와 통신을 통해 클라이언트가 요청한 데이터 반환
